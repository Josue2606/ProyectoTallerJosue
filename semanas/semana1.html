📚 Semana 1 - Desarrollo de Aplicativos

Ejercicios básicos en Java con Swing: Interfaces gráficas y manejo de eventos.

Descripción General

En esta semana, creé dos aplicaciones de escritorio en NetBeans:

  • 🔹 Calculadora de Inversión: Calcula el costo total para una empresa en una feria (alquiler de stand, publicidad, personal).
  • 🔹 Descuento en Camisas: Determina descuentos basados en la cantidad comprada y muestra el monto final.

Objetivo: Introducción a JFrame, componentes Swing y eventos de botones.

Detalles de los Ejercicios

Calculadora de Inversión: El usuario ingresa valores numéricos en campos de texto. Al clic en "Calcular", suma los costos y valida entradas (manejo de excepciones).

Descuento en Camisas: Reglas: 1-5 camisas (sin descuento), 6-10 (10%), >10 (20%). Muestra total con descuento aplicado.

Ambos usan GridLayout para la interfaz y ActionListener para eventos.

Ejemplo de Código - Calculadora de Inversión

// Calculadora de Inversión - Semana 1 // Autor: Josue Anthony Arredondo Condor // Uso: NetBeans con Swing import javax.swing.*; import java.awt.*; import java.awt.event.*; public class CalculadoraInversion extends JFrame { private JTextField txtAlquiler, txtPublicidad, txtPersonal; private JButton btnCalcular; private JLabel lblTotal; public CalculadoraInversion() { setTitle("Calculadora de Inversión - Feria Comercial"); setSize(350, 250); setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); setLayout(new GridLayout(5, 2, 10, 10)); // Componentes add(new JLabel("Alquiler de Stand (S/):")); txtAlquiler = new JTextField(); add(txtAlquiler); add(new JLabel("Publicidad (S/):")); txtPublicidad = new JTextField(); add(txtPublicidad); add(new JLabel("Personal (S/):")); txtPersonal = new JTextField(); add(txtPersonal); add(new JLabel("")); btnCalcular = new JButton("Calcular Total"); add(btnCalcular); lblTotal = new JLabel("Total: S/ 0.00"); add(lblTotal); // Evento del botón btnCalcular.addActionListener(new ActionListener() { @Override public void actionPerformed(ActionEvent e) { try { double alquiler = Double.parseDouble(txtAlquiler.getText()); double publicidad = Double.parseDouble(txtPublicidad.getText()); double personal = Double.parseDouble(txtPersonal.getText()); double total = alquiler + publicidad + personal; lblTotal.setText("Total: S/ " + String.format("%.2f", total)); } catch (NumberFormatException ex) { lblTotal.setText("Error: Ingresa solo números válidos"); } } }); setVisible(true); } public static void main(String[] args) { SwingUtilities.invokeLater(() -> new CalculadoraInversion()); } }

Nota: Código simplificado. El ejercicio de Descuento en Camisas sigue una estructura similar.
